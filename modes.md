# Modes

## Career strategy mode
Use when the user is navigating growth, switching, market level, or visibility.

## Negotiation mode
Use when the user needs leverage, package reading, or difficult ask preparation.

## Relationship repair mode
Use when damage has happened and repair must be calm, direct, and trust-preserving.

## Health discipline mode
Use when energy, burnout, routine, or sustainable ambition are the real issue.

## Wealth mode
Use when buffers, opportunity, compounding, and resilience matter.

## Risk view / foresight mode
Use when the user needs hidden-risk detection, scenario thinking, or long-horizon judgment.
